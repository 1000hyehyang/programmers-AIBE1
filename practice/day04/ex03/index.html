Flexbox 완벽 가이드

Flexbox란 무엇인가?

Flexbox는 유연하고 반응형 레이아웃 구조를 쉽게 설계할 수 있도록 도와주는 CSS 레이아웃 모듈입니다.

Flexbox 주요 속성

Flexbox는 컨테이너와 아이템에 다양한 속성을 설정하여 레이아웃을 제어할 수 있습니다.

  • 컨테이너 속성: display, flex-direction, justify-content, align-items, flex-wrap, align-content
  • 아이템 속성: flex-grow, flex-shrink, flex-basis, flex (축약형), order

justify-content와 align-content의 차이

justify-content: 주축(main axis)에서 아이템을 정렬합니다. 아래 예제에서는 아이템이 주축을 따라 간격을 두고 배치됩니다.

align-content: 여러 행(row)을 교차축(cross axis)에서 정렬합니다. 아래 예제에서는 행 간의 간격을 조정합니다.

Flex 아이템 속성 예제

Flex 아이템 속성을 활용하여 개별 아이템의 크기와 동작 방식을 제어할 수 있습니다.

flex-grow, flex-shrink, flex-basis
Grow 2
Grow 1
Grow 0

위 예제에서:

  • flex: 2 1 100px: 첫 번째 아이템은 나머지 아이템보다 2배 더 많은 공간을 차지합니다.
  • flex: 1 1 100px: 두 번째 아이템은 기본 공간을 사용하고, 남은 공간의 1배를 차지합니다.
  • flex: 0 1 100px: 세 번째 아이템은 남은 공간을 차지하지 않습니다.

정렬 조합 예제

align-items와 justify-content를 함께 사용하여 교차축과 주축에서 아이템을 정렬할 수 있습니다.

align-items와 justify-content 조합

위 예제에서는:

  • justify-content: center: 주축에서 아이템이 가운데로 정렬됩니다.
  • align-items: flex-end: 교차축에서 아이템이 아래쪽으로 정렬됩니다.
