CSS Position 예제

Fixed Position

요소가 뷰포트를 기준으로 고정되어 스크롤해도 위치가 변하지 않습니다.

position: fixed; top: 10px; right: 10px;

Sticky Position

요소가 일반적인 흐름에 따라 배치되다가 지정된 임계점에 도달하면 고정됩니다.

position: sticky; top: 0;

Absolute Position

요소가 가장 가까운 위치 지정 조상을 기준으로 배치됩니다.

position: absolute; top: 20px; left: 20px;

Relative Position

요소가 자신의 원래 위치를 기준으로 이동합니다.

position: relative; top: 10px; left: 10px;

Static Position

요소가 일반적인 문서 흐름에 따라 배치됩니다.

position: static;
